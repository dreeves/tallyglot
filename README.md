Bookmarklet that shows the wordcount of what you're composing.
Works on many web pages.
If it doesn't work on a page you care about, let me know.
It's easy to add more.

And it gives you a pretty straightforward way to control what actually counts towards your wordcount.

Further exposition [on LessWrong](https://www.lesswrong.com/posts/8G24bWJ5nE2QjCGAb/see-your-word-count-while-you-write).

Built as part of [Inkhaven](https://inkhaven.blog).
